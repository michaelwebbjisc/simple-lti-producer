LtiLibrary is designed to make it easy to build IMS LTI Tool Consumers and Tool Providers with ASP.NET.
This version of LtiLibrary support LTI 1.0, 1.1, 1.1.1 and draft 1.2.

LtiLibrary is available via NuGet (https://www.nuget.org/packages/LtiLibrary/). There is a also a sample
package in NuGet (https://www.nuget.org/packages/LtiLibrary.Sample).

To get started with the sample:

1. Start a new ASP.NET Web Application.
2. Select the Empty template.
3. Add folders and core references for Web Forms, MVC and Web API.
4. Use the Package Manager to install the LtiLibrary.Sample package.
5. Run the application.
